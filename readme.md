<p>A/Left arrow: Left</p>
<p>D/Up arrow: Right</p>
<p>C/J: Dash</p>

<p>Y: Disable screen refresh</p>
<p>O: Slow motion</p>
<p>G: Debug screen</p>
<p>N: Invisible mode</p>
<p>M: Secret function</p>
<p>`: Random inputs (good luck)</p>
